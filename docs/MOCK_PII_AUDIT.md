# Mock 个人数据 — 全量清单与替换建议

> **日期**：2026-05-19 ｜ **作者**：architect
> **触发**：用户反馈页面上 mock 用了"真实个人姓名"

---

## 1. 核心发现

**真正需要替换的只有 1 个值：`*宇澄`（疑似真实姓名"宇澄"加星号脱敏）**，散布在 **6 个文件 7 处**。

其他"敏感格式"的 mock（身份证号、手机号、银行账户）都已是**格式合规的脱敏假数据**，未泄露具体个人信息——但如要"演示阶段一并清理为典型假数据"，也一并替换更整洁。

后端 prompt / 后端代码**没有含个人姓名的 mock**——填充值都是非个人的（如 "本人"、"4800"、"2026"）。

## 2. 个人姓名 `*宇澄` / `**澄`（7 处）

| # | 文件:行 | 当前 | 上下文 |
|---|---|---|---|

| 4 | `app/lib/pages/yibao_query_page.dart:102` | `'*宇澄'` | 医保查询 — 账户信息卡 |

| 7 | `app/lib/pages/wireframe_page.dart:567` | `'**澄'` | wireframe 文档占位（论文示意图） |

**建议统一替换值**：

**选项 A（推荐）**：`*宇澄` → **`*三`** （三个字，"张三"通用假名脱敏成"姓 + 星 + 名"中常用的"星 + 名"形式；与 `**澄` 同长度的替换是 `**三`）

**选项 B**：`*宇澄` → **`*小明`**（与原长度一致，"王小明"脱敏）

**选项 C**：`*宇澄` → **`*建国`** （更具老年用户感）

建议 B，演示和论文场景"王小明"识别度最高、毫无歧义。

**对应替换**：
- `'*宇澄'` → `'*小明'`（5 处）
- `'**澄'` → `'**明'`（2 处）

## 8. 完整替换 sed 命令（开发参考）

如果开发要一次性替换"宇澄"为"小明"：

```bash
cd /home/getui/codes/liangyc/archive/test/app/lib/pages
# 5 处 '*宇澄' → '*小明'
sed -i "s/'\*宇澄'/'*小明'/g" \
  mine_page.dart shebao_query_page.dart pension_query_page.dart \
  yibao_query_page.dart shebao_jiaona_page.dart

# 2 处 '**澄' → '**明'
sed -i "s/'\*\*澄'/'**明'/g" face_auth_page.dart wireframe_page.dart
```

或者**手动逐行编辑**（更安全）：开发用 IDE 全局 search & replace，先确认每处上下文再改。
